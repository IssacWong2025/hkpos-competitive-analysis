# 市场研究数据字典

本目录包含所有市场占有率数据收集的原始数据和说明。

## 文件命名规范
- 文件名格式: `competitor_<data_type>.xlsx`
- 所有文件使用 UTF-8 编码
- 日期格式: YYYY-MM-DD

## 数据收集原则
1. **证据优先**: 所有关键数据点必须提供来源链接或截图
2. **时间标注**: 每个数据集必须注明统计日期
3. **置信度**: 标注数据可靠性（高/中/低）
4. **可追溯**: 保留查询历史和变更记录
